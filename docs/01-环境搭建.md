# 01 - 环境搭建

## 前置要求

- Node.js ≥ 20
- pnpm ≥ 9
- 微信开发者工具（小程序调试）
- VS Code + Vue Official 插件

## 安装步骤

```bash
cd /Users/martin/Documents/Github-uniapp/my-poly
pnpm install
```

## 启动命令

| 命令 | 说明 |
|------|------|
| `pnpm dev:mp` | 微信小程序开发 |
| `pnpm dev:h5` | H5 开发（http://localhost:9000） |
| `pnpm build:mp` | 小程序生产构建 |
| `pnpm type-check` | TS 类型检查 |

## 微信开发者工具

1. 运行 `pnpm dev:mp`
2. 导入 `dist/dev/mp-weixin`
3. AppID 使用：`wx41143627eaf3541f`

## 难点

- uni-app 小程序端 `dev` 实际走 `build development` 模式，环境变量以 `--mode` 为准，不是 `command === serve`
- 微信 CLI 自动打开需配置 `WECHAT_DEVTOOLS_CLI_PATH`（非默认安装路径时）

## 易错点

1. **必须用 pnpm**：项目 `preinstall` 限制了包管理器
2. **导入目录错误**：应导入 `dist/dev/mp-weixin`，不是项目根目录
3. **sass 版本**：Wot UI v2 需要 `sass@^1.98.0`，过低会编译失败
4. **husky 初始化**：首次 `pnpm install` 会跑 `prepare`，若 git 未初始化可能告警，可 `git init` 后重试

## 注意点

- 本地 API 调试时，微信开发者工具需勾选「不校验合法域名、web-view、TLS 版本」
- H5 开发已启用代理（`.env.development`），避免跨域

## 适用场景

- 新成员 onboarding
- CI/CD 环境配置参考
- 换机重装开发环境
